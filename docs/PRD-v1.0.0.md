# 产品需求文档 (PRD) - Lucas Space 个人品牌全栈网站

## 1. 文档基本信息
* **项目名称**: Lucas Space (刘俊雄个人品牌全栈网站)
* **创建时间**: 2026年6月
* **项目负责人**: 刘俊雄 (Lucas)
* **当前版本**: v1.0.0
* **核心定位**: 融合“7年前端经验高光”与“全栈独立博客”的数字化职业名片

---

## 2. 核心目标与产品价值
* **数字化简历**: 将传统 PDF 简历立体化，通过公网域名直观展示技术实力。
* **技术炫技落地**: 证明在 Nuxt 3 (SSR)、Tailwind CSS、工程化、自适应布局以及全栈接口方面的落地能力。
* **极致用户体验**: 提供极致的首屏加载速度（LCP < 1.2s）与纯正深色极客风界面。

---

## 3. 技术选型架构
本项目彻底舍弃复杂的国外英文动效库与传统后端服务器开销，采用以下高内聚、轻量化的全栈架构：
* **全栈前端框架**: Nuxt 3 (基于 Vue 3 生态，开启 SSR 服务端渲染)
* **样式/响应式**: Tailwind CSS (实现 100% 移动端/PC 端视网膜屏自适应)
* **动效层**: Vue 3 原生 `<Transition>` & `<TransitionGroup>` + Tailwind 动画类（全中文文档生态，0kb 额外体积负载）
* **数据层（博客）**: Nuxt 3 全栈服务器接口 (`server/api/`) 路由模拟与读取
* **互动留言层**: Giscus 组件（基于 GitHub Discussions API，实现零数据库成本的高安全留言板）
* **托管/服务器平台**: Vercel (Serverless 无服务器架构，享受全球 CDN 加速与全自动 CI/CD)

---

## 4. 页面布局与功能模块划分

网站在桌面端（PC）采用 **左侧固定名片、右侧上下分栏滚动** 的现代极客布局；在移动端（手机）自动转为 **垂直单页流式布局**。

### 模块 A：左侧聚合名片区 (`AvatarCard.vue`)
* **用户头像与基本信息**: 展示圆形个人头像，名字、Title（高级前端开发工程师 | 7年经验）。
* **动态求职状态**: 包含带脉冲动画的标签：🟢 `正在寻找新机会 | 深圳`。
* **社交矩阵链接**: GitHub、简书博客、个人邮箱的跳转入口。
* **核心技能树卡片**: 将 7 年技能分类固化展示：
  1. *框架与核心*: Vue3/Pinia、React/Hzer、TypeScript
  2. *工程化与性能*: Nuxt.js (SSR官网优化)、Vite/Webpack、Jenkins 自动化
  3. *大屏与图表拖拽特长*: ECharts、jsplumb（思维导图）、可视化低代码拖拽
  4. *前沿生产力*: Cursor、Claude Code、Vibe Coding
* **简历一键下载**: 点击按钮直接触发公网下载 `public/resume-lucas.pdf`。

### 模块 B：右上博客中心区 (`BlogCard.vue` & `server/api/blog.ts`)
* **文章卡片流**: 以卡片形式展示近期发表的技术文章。
* **数据仿真与高光展示**: 结合真实的金融 AI 与可视化项目经验，拟定核心亮点文章（如《基于 jsplumb 和大模型的思维导图架构实践》）。
* **核心功能指标**: 包含阅读时长预估（e.g., `13 min read`）、分类标签（#SEO、#可视化）。

### 模块 C：右下互动留言板 (`Guestbook.vue`)
* **免库安全留言**: 集成 Giscus 插件。
* **防灌水鉴权**: 用户必须通过标准的 GitHub OAuth 授权才能发表留言，从根源上杜绝恶意刷票行为。
* **样式契合**: 留言板的外观（Dark/Light 模式）必须与个人站的深色极客风完全暗合。

---

## 5. 非功能性需求 (服务器与性能指标)
* **零服务器维护成本**: 项目通过 `vercel.json` 宣告规则，完全交由 Vercel 托管为 Serverless 架构，不租用任何云服务器，零基础续费开销。
* **极致的首屏性能 (SEO 优化)**: 利用 Nuxt 3 的混合渲染 (Hybrid Rendering)，首页和静态资源强制在编译期生成 HTML（Static），保证加载耗时极低。
* **源码级动画控制**: 全站进场动画和标签悬停微动效均通过 Vue 3 原生指令与生命周期钩子控制，无第三方大型 JS 动画库阻碍线程，确保滚动顺畅不卡顿。
